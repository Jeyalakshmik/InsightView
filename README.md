🚀 InsightView Dashboard
🔗 Live Demo: https://insight-view.vercel.app

📌 Project Description
**InsightView** is an interactive analytics dashboard built using Next.js and modern UI libraries. 
It allows users to visualize data through dynamic widgets such as charts, KPIs, and tables. 
The platform is designed to provide real-time insights and an intuitive drag-and-drop interface for customizing dashboards.

✨ Features
- 📊 Interactive dashboard with charts and KPIs  
- 🎨 Drag-and-drop widget system  
- 📱 Fully responsive design  
- 🤖 AI-powered insights (Get AI Summary)  
- ⚡ Fast performance using Next.js  
- 🔥 Modern UI with Tailwind CSS
