# **App Name**: InsightView Dashboard

## Core Features:

- Customizable Interactive Dashboard: Allows users to dynamically arrange, resize, and configure a variety of data visualization widgets (charts, tables, KPI cards) on a grid canvas to create personalized dashboards.
- Dynamic Widget Configuration: Enables users to customize specific parameters for each widget, including chart types, metrics, aggregation methods for KPIs, and data fields for tables and charts, through dedicated settings panels.
- Customer Order Management: Provides a comprehensive module for creating, viewing, editing, and deleting customer order records, feeding the dashboard with relevant data.
- Global Date Filter: Implements a dropdown filter that allows users to select date ranges (e.g., 'Last 30 Days', 'All Time') to instantly update the data displayed across all dashboard widgets.
- Responsive Grid System: Ensures the dashboard layout and widgets automatically adapt to different screen sizes (desktop, tablet, mobile) with optimized grid columns and widget stacking for a seamless user experience.
- Persistent Dashboard Layouts & Data: Allows users to save their customized dashboard layouts and widget configurations, as well as storing all customer order data, for retrieval and continued use.
- AI Order Trend Summarizer Tool: An AI tool that analyzes the customer order data currently displayed on the dashboard (filtered by date) and generates a concise summary of key trends, anomalies, or important insights.

## Style Guidelines:

- Primary brand color: A calm, confident blue (#2268CC) conveying clarity and reliability for key interactive elements and data representation.
- Background color: A very light, desaturated blue (#ECF2FB) to provide a clean and expansive canvas that allows data visualizations to stand out without distraction.
- Accent color: A vibrant teal-cyan (#1CBFDB) for highlighting specific metrics, notifications, or call-to-action buttons, offering a fresh contrast.
- Headline and body text font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly legible appearance across all data and textual content.
- Utilize a consistent set of clean, vector-based icons that clearly represent dashboard actions (e.g., drag, delete, settings) and widget types for intuitive navigation and functionality.
- Implement a structured grid-based layout for the dashboard canvas, emphasizing modularity and clear hierarchy for widgets and data panels. Utilize modal dialogues for focused configuration tasks.
- Incorporate subtle, smooth animations for drag-and-drop operations, widget resizing, and data updates to enhance responsiveness and user feedback, improving the perceived performance and fluidity.
